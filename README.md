# Counter

> Rewrite Counter in Kotlin Multiplatform

## Introduction

When you click the button, the number will increase by 1.

## Basic Info

### Platform

- [x] Android
- [x] Desktop

### Features

- [x] Save the number to the database
- [x] Dark mode
- [x] Monet color (Android only)
